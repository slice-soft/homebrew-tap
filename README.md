# homebrew-tap

Homebrew tap for SliceSoft tools. Provides formula to install the [Keel CLI](https://github.com/slice-soft/keel) on macOS and Linux.

## Usage

```bash
brew tap slice-soft/tap
brew install keel
```

## Available formulas

| Formula | Description |
|---------|-------------|
| `keel`  | Opinionated Go framework CLI for building structured services |

## Requirements

- macOS (Intel or Apple Silicon) or Linux (amd64 / arm64)
- [Homebrew](https://brew.sh)
